# stage_4_developer_roadmap-
CSS3 (каскадные таблицы стилей)

* Блочная модель: content, padding, border, margin, box-sizing: border-box.
* Селекторы: по тегу, классу, id, атрибуту, псевдоклассы (:hover, :focus, :nth-child), псевдоэлементы (::before, ::after).
* Специфичность и каскад (почему одни стили переопределяют другие).
* Единицы измерения: px, %, em, rem, vh, vw. Принципиальная разница между em и rem.
* Позиционирование: static, relative, absolute, fixed, sticky.
* Flexbox: display: flex, justify-content, align-items, flex-direction, flex-wrap, gap.
* Grid: display: grid, grid-template-columns/rows, gap, fr.
* Адаптивная вёрстка: @media, min-width/max-width, mobile-first подход.
* Базовые переходы: transition, transform.
* CSS-переменные: :root { --color: #333; }, var(--color).


```px```	Фиксированные пиксели экрана	Границы, тени, мелкие отступы
```%```	Родительского элемента	Ширина колонок, адаптивные блоки
```em```	Размера шрифта текущего элемента	Отступы внутри компонента
```rem```	Размера шрифта корневого элемента (<html>)	Глобальные отступы, размеры текста
```vh```	1% от высоты окна браузера	Высота полноэкранных блоков
```vw```	1% от ширины окна браузера	Адаптивные шрифты
